Media Keys
----------
Added support for media keys on mac keyboards. 

| Key              | Function          |
| ---------------- | ----------------- |
| PlayPause (F8)   |  Plays/Pauses     |
| Rewind(F7)       |  Skip to Previous |
| Fast forward(F9) |  Skip to Next     |
